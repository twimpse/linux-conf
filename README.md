# linux-conf
This is a setup script I use as a basic setup for debian based linux servers. 

It adds some basic packages and changes some default settings. Does some basic 
tightening and hardening of the system. Nothing advanced or special, just restricts
some stuff that is (too) permissable from the start. 

If you are not me, DO NOT, let the script add the ssh-key to your account. If you
do, you are giving me access to your account. You probably don't want that. 

Actually, you probably don't need this script at all. There is nothing really special
about it. I use it to save time (about 20-30 minutes) when I first install a
linux system. And it might not be changes that you want or need. 

Run the script at your own risk. But the script can be used as a reference to see
which things may need adjusting on a vanilla system.
